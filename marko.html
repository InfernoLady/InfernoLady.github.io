Tko su programeri?

Programer je osoba koja se bavi razvojem softvera , uključujući istraživanje, dizajn, programiranje i testiranje računalnih programa.
Software developer ima zadaću da:

  Isplanira
  Dizajnira
  Programira
  Te analizira program za potrebe korisnika

Planiranje samog softwera je jako važan dio samog procesa jer u tom koraku korisnik daje ideju i smjernice što bi njegov krajnji rezultat htio biti i programer može prepoznati nepotpune i nejasne dijelove samog softwera i dati svoje osobne prijedloge.

Popularni programski jezici

Dizajniranje je korak u kojemu programer stvar osnovni model softwera s njegovim osnovnim dijelova koji će se koristi npr. Programski jezik, operativni sustav, hardwerske komponente.
Samo programiranje je naravno najzabavniji korak samog procesa jer programer u tom koraku primjenjuje sve svoje znanje i vještine u pisanju koda.

Neki od programskih jezika koje se danas koriste su:
  Java-objektno orijentirani programski jezik baziran na C++, ali sa pojednostavljenom sintaksom i pojednostavljenom kontrolom memorije
  Osnovni koncept u Javi je klasa tj. sav izvorni kod je napisan unutar klasa.

  C je programski jezik opće namjene s podrškom za objektno orijentirano programiranje
  C++ je produženi C koji se prvo zvao 'C s klasama' a kasnije dobio svoje današnje ime C++
  C# je izumljen s ciljem da .NET platforma dobije programski jezik, koji bi maksimalno iskoristio njezine sposobnosti. Sličan je programskim jezicima Java i C++
  Python- dopušta programerima korištenje nekoliko stilova programiranja: objektno orijentirano, strukturno i aspektno orijentirano programiranje. Ova fleksibilnost čini python programski jezik sve populariniji.
  Ruby i Php
